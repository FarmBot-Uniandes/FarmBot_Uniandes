1. FarmBot Simulator: https://github.com/FarmBot-Uniandes/Simulator
2. FarmBot Controller: https://github.com/FarmBot-Uniandes/Controller
3. FarmBob WebApp: https://github.com/FarmBot-Uniandes/WebApp
